# GoDumpLsass


GoDumpLsass is a simple tool that can dump lsass without to get caught by Windows Defender.
## Releases

https://github.com/Enelg52/GoDumpLsass/releases

## Usage

```txt
.\GoDumpLsass.exe
```

## Example
```txt
.\GoDumpLsass.exe
[*] Get process id
[*] Open lsass handle
[*] Dump lsass
```
